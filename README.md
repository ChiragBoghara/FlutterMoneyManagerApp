# My Money Manager

A new Simple money manager app to track daily income and expenses within few seconds.

<h2>Description</h2>
<p>If you have habit of keeping an eye on incomes and expenses then this app is for you. Inside the app, you will find a simple User Interface that allows you to add new incomes or expenses along with assigning the transaction with fancy icon to indicate its category. You can add new transaction by pressing + iconbutton at bottom. You can show all the details of transaction by just clicking on the transaction. You can update transaction by long press. You can also add your custom category from categories screen in drawer.It has been designed in an intuitive way, quick to add, easy to navigate.</p>

<p>You can also set app accent color that you want..</p>
